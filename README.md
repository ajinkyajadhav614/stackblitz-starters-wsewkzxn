## 🚀 Deploy in 10 Minutes (Free)

### Step 1 — Create a GitHub Account
1. Go to **https://github.com**
2. Click "Sign up" and create a free account

### Step 2 — Upload this Project to GitHub
1. Click the **+** button → "New repository"
2. Name it: `ashirwad-crm`
3. Click "Create repository"
4. Click "uploading an existing file"
5. Drag and drop ALL files from this folder (including `src/` and `public/` folders)
6. Click "Commit changes"

### Step 3 — Deploy on Vercel (Free Hosting)
1. Go to **https://vercel.com**
2. Click "Sign up" → choose "Continue with GitHub"
3. Click "Add New Project"
4. Select your `ashirwad-crm` repository
5. Click **Deploy** (no settings to change)
6. Wait 2 minutes ⏳
7. ✅ Your CRM is live! Vercel gives you a URL like:  
   `https://ashirwad-crm.vercel.app`

## 📱 How to Use

| What you want | How to do it |
|---|---|
| Open CRM | Go to your Vercel URL on any phone/computer |
| Add customer | Customers tab → + Add Customer |
| New application | Services tab → + New Application |
| Record payment | Payments tab → + Record Payment |
| View receipt | Payments tab → 🧾 View |
| Follow-up reminders | Follow-ups tab |
| Customer self-service | Kiosk tab |
| Add/remove service types | Services tab → left panel |

---

## 👥 User Roles

| Role | Access |
|---|---|
| Super Admin | Full control |
| Admin | Customers, services, payments, reports |
| Operator | Create applications, upload docs |
| Receptionist | Register customers, view status |
| Customer | Kiosk portal only |

## ⚠️ Important Note

This version stores data **in the browser only** (no database).  
Data will reset if you clear your browser or open on a different device.

**To save data permanently**, the next step is connecting a free database (Supabase).  
Ask your developer or contact us for help with that.
